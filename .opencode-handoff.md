## Arquitectura actual
- **Frontend:** React SPA con Router casero (`useSyncExternalStore`), `ShopPage` con dos vistas (`home`/`shop`), estado global via `CartContext` + `AuthContext` con persistencia en `localStorage`
- **Lógica:** Servicios puros en `src/services/`, hooks en `src/hooks/`, tipos compartidos en `src/types/`
- **Datos:** 17 productos en `data/db.json` (cargados vía `fetch` directo)

## Testing
- **Unit tests:** 70 tests (Vitest + jsdom) — products.service (14), cart (13), checkout (24), auth (8), concierge (11)
- **E2E tests:** 14 tests (Playwright) — home, shop, cart, navigation flows

## Archivos clave
- `src/App.tsx` — Entry point + router + page components
- `src/services/productSearch.ts` — Índice TF-IDF + cosine similarity para RAG local del concierge
- `src/hooks/useConcierge.ts` — Lógica de conversación y parseo de intención del chatbot
- `src/components/chat/ShoppingConcierge.tsx` — UI del chatbot (FAB + panel + burbujas)
- `data/db.json` — 17 productos en formato json-server
- `server/` — Backend real Express + TypeScript + SQLite
